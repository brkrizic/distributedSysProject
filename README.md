IoT Sensor Dashboard
This project is a distributed IoT sensor monitoring system built with Python, FastAPI, and Next.js.

Sensors: Supports ESP32 microcontrollers or Python simulations for live sensor data.
Backend: FastAPI server collects data from multiple sensors and exposes a REST API.
Frontend: Next.js dashboard visualizes live sensor readings in real-time tables, graphs, and maps.
Features:
Real-time data updates every 2 seconds
Interactive map with sensor locations
Live temperature and GPS tracking
Responsive, professional dashboard design
